# シナリオ
----------
## 商品を補充する
#####アクター : 管理者
#####事前条件 : 
#####トリガー : 前回の補充から一定期間が経過
```txt
1. 管理者が商品の残りを確認する
2. 管理者が足りない商品を補充する

```
## 商品を購入する
#####アクター : 顧客
#####事前条件 : 
#####トリガー : 顧客が商品を買おうと思う
```txt
1. 顧客がお金を入れる
2. 顧客が商品を選択する
3. 取り出し口に商品が出る
```

## 商品を管理(保温・冷蔵)する
#####アクター : 温度管理システム
#####事前条件 : 
#####トリガー : 前回の管理のアクションから一定時間経過
#####主シナリオ :
```txt
1. 温度管理システムが現在の温度を読み取る
2. 現在の温度と設定温度の差を計測する
```
#####副シナリオ :
```txt
2a : 設定温度との差がある場合、冷蔵を行う
```
